# yuzoiwasaki.me

[yuzoiwasaki](https://github.com/yuzoiwasaki) のポートフォリオサイトです

## Developing


```bash
npm run dev

# or start the server and open the app in a new browser tab
npm run dev -- --open
```

## Building

```bash
npm run build
```

You can preview the production build with `npm run preview`.
